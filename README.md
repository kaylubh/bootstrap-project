# bootstrap-project

This is a Bootstrap project for a fictional county government website. The project uses Bootstrap 5 for styling and layout, and includes custom CSS for additional styling.

## Attribution

This project was created by Caleb Hemphill for Wake Technical Community College's Web 210 course. It is intended for educational purposes only and is not affiliated with any real government entity. All content and design elements are original and created by the author.
